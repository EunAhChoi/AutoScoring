Auto Scoring

Automatically score with the test paper provided! Automatic scoring can be done with a click.

Details

Manual

1. Receive students' test paper from the dropbox (optional)

2. Select the test paper type. (numeric/english word)

3. Select the blank test paper via the file selection button

4. Select a test paper with correct answers

5. Select all student test papers you want to grade

6. If you press the Grading button, all grading results will be saved in the local folder.
